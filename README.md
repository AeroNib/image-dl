# image-dl

Python CLI for downloading images from a website.

## Installation

Install with pipx:

```bash
pipx install git+https://github.com/AeroNib/image-dl.git
```

Or install from a local clone:

```bash
pipx install .
```

To upgrade an existing pipx installation to the latest version:
```bash
pipx upgrade image-dl
```
